This is a way to cool web server, let's make some test!

Yes, they work "out of the box"

The Connect form Senchalabs do all the refreshing needed for a server. Pure node don't do that (at least for me). It can be installed by npm

    
    npm install connect

Of course, Node.js it has to be allready installed, It supplies build in repositories for major OS's as you can see on Joyent Github page. If you are using Ubuntu, as i do, you can folow the next lines.

    
	sudo apt-get install python-software-properties
	sudo add-apt-repository ppa:chris-lea/node.js
	sudo apt-get update
	sudo apt-get install nodejs npm

Now you can install connect by npm

For getting it running, make a test.html with some content

Hello From Node.js!

then the actually node.js server script as node.js containing

    
		var connect = require('connect');
	connect.createServer(
		connect.static(__dirname)
	).listen(8080);

From console browse to where the folder containing these two files is and run node node.js

In the final step you should point your browser to http://127.0.0.1:8080/test.html

And you should read the greetings!
